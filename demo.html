AlteR. Complete demo.

AlteR. Gestures.

This video demonstrates the use of the different gestures available for manipulating your model using AlteR .

AlteR. Persistence.

This video demonstrates how you can save and load a model in AlteR.

AlteR. References.

This video demonstrates how you can add references between elements in AlteR. The link can have different colors and width.

AlteR. Code scanner.

This video shows AlteR used for a case study in inventory, where the information about devices (computers, screens, printers) is retrieved from an information system and is displayed using AR overlay.

AlteR. Ibeacons detector.

This video shows AlteR used for a case study in post-its, where shows messages, images or videos using AR overlay. Also there is a configuration tool inside that allows adding your own codes and Ibeacons

AlteR. Augmented Home.

This video shows AlteR used for a case study in Augmented Home. Now you can choose also images like a picture and see local weather

AlteR. Museum.

This video shows AlteR used for a case study in Museum. It use QR code scanning to load explanatory videos

AlteR. Telecom.

This video shows AlteR used for a case study with virtual network, with the main functionality of the app
